# vue2-element-project

## 项目简介
这是一个基于 Vue 2 和 Element UI 的项目，旨在提供一个简单的用户界面，包含登录和主页功能。

## 技术栈
- Vue 2
- Element UI
- Vue Router
- Vuex
- SCSS

## 安装与运行
1. 克隆项目：
   ```bash
   git clone <repository-url>
   ```
2. 进入项目目录：
   ```bash
   cd vue2-element-project
   ```
3. 安装依赖：
   ```bash
   npm install
   ```
4. 启动开发服务器：
   ```bash
   npm run serve
   ```
